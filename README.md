## POPCAT
同時我也響應了POPCAT活動用JS寫了一個簡單的聯點器  
這是FOR無限迴圈版  
另外還有[其他的](https://github.com/banahaker/PopcatJSsupport/blob/main/popcat.js)
```javascript
var event = new KeyboardEvent('keydown', {  
    key: 'L',  
    ctrlKey: true  
});  
let limit = 10;  
for (var i = 0; i < limit; i++) {  
    document.dispatchEvent(event);  
    limit++;  
}  
```
